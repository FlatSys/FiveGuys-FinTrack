# "FinTrack" – Personal Finance & Budget Forecaster
**Project Outline**

Students often struggle to manage finances. We have built a smart budgeting tool that not only tracks spending but predicts future financial health.

## Features
- Transaction Manager: Add, View, and Categorize income and expenses.
- Smart Forecasting: A function that projects the user's balance for the next 30 days based on their current balance and RecurringBill objects.
- Budget Alerts: Users set a budget for a category (e.g., "Food: £200"). The system must warn the user if a new expense pushes them over this limit.
- Reporting: Generate a summary showing "Needs vs. Wants" spending ratios.
- Data Visualization. Provides a visual representation of spending.
